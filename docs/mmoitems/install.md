# 🔌 Installation Guide

## Installing MMOItems Premium

<p>Drag and drop the latest builds of MythicLib and MMOItems on your server and restart it.</p>

## Addon: MMOMana

<p>This addon was replaced by the MythicLib <a href="./../mythiclib/misc/mana-stamina">native mana/stamina module</a>.</p>

## Addon: MI PerWorldDrops (<a href="https://github.com/mmopluginteam/mmoitems-perworlddrops/releases" target="_blank" rel="noreferrer">Download</a>)

<p>MMOItems PerWorldDrops allows you to have <strong>drop tables restricted to certain worlds</strong> for blocks and monsters. You can drag &amp; drop this addon in your plugins folder just like with MMOItems. A /drops folder should appear in the MMOItems plugin folder after rebooting your server. In that folder, each .yml you create corresponds to a specific world (the file name has to match <strong>EXACTLY</strong> the world name). In order to know how to setup these YAML configs, please refer to this <a href="./items/drop-tables">wiki page</a>.</p>
<p>This addon is kept up to date and comes handy if you don't need huge/complex drop table plugins like MMOCore or MythicMobs.</p>
