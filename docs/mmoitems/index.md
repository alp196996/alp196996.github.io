<p>MMOItems is the last RPG item plugin you will ever need. Create custom weapons, items, armors, consumables... with unique stats, display options and abilities. It is fully compatible with the most popupar resource pack management plugins such as ItemsAdder, Nexo or Oraxen. MMOItems can use skills from MythicMobs and features RPG functionalities like item identification, item tiers, item sets, reforging, crafting stations, Soulbound, and many many more.</p>
