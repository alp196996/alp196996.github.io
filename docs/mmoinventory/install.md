# 🔌 Installation Guide

<ul>
  <li>
    Download and install
    <a href="https://www.spigotmc.org/resources/mmolib-mythiclib.90306/" target="_blank"
    rel="noreferrer">
      MythicLib
    </a>
    . It is the only hard dependency for MMOInventory.
  </li>
  <li>
    Download and install the MMOInventory JAR file in the
    <code>
      /plugins
    </code>
    folder.
  </li>
  <li>
    We recommend reading
    <a href="./compatibility/mmoitems">
      this wiki page
    </a>
    if you are running MMOItems on your server.
  </li>
  <li>
    Download the default MMOInventory server resource pack and install it
    (or merge it with your existing server pack).
  </li>
</ul>
