# 💾 Commands

<table tabindex="0">
  <thead>
    <tr>
      <th>
        Command
      </th>
      <th>
        Description
      </th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        <code>
          /mmoprofiles reload
        </code>
      </td>
      <td>
        Reloads the plugin and config files. Watch out, some options require a
        server RESTART.
      </td>
    </tr>
    <tr>
      <td>
        <code>
          /mmoprofiles open &lt;player-name&gt;
        </code>
      </td>
      <td>
        Opens profile selection for a player. This should only be used when the
        <code>
          no-gui-on-login
        </code>
        option is toggled on.
      </td>
    </tr>
    <tr>
      <td>
        <code>
          /mmoprofiles export-data
        </code>
      </td>
      <td>
        Exports player data from flat storage to SQL
      </td>
    </tr>
  </tbody>
</table>
