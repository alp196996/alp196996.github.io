# 😉 When Logging In

<p>
  When a player logs in, they become invincible and are forced to select
  a profile in the Profiles GUI (see next page). Once they have chosen their
  profile, all the data from that profile will be loaded and applied to the
  player. The player can create or remove profiles in the GUI, but they are
  limited to a certain amount of simultaneous profiles. This can be fully
  customized using the rank system (more here).
</p>
<p>
  Here is an extensive list of all the vanilla features that are saved and
  applied between profiles:
</p>
<ul>
  <li>
    Health, Food level and saturation
  </li>
  <li>
    Air Level
  </li>
  <li>
    Level &amp; Experience
  </li>
  <li>
    Last Location &amp; Bed Spawn Location
  </li>
  <li>
    Inventory, Equipment &amp; Ender chest
  </li>
  <li>
    Gamemode
  </li>
  <li>
    Potion Effects
  </li>
  <li>
    Vanilla Attributes
  </li>
</ul>
