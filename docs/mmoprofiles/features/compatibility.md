# 🦾 Compatibility

<p>
  The main feature of this plugin is to enable other plugins to use its
  API and also have profile-based data. A plugin that supports MMOProfiles
  will have its data saved for the profile and not for the player, and changing
  profile will be as if you logged in with a different account.
</p>

## Vault Economy

<p>
  MMOProfiles saves the player's Vault balance. This way, any Vault-based
  economy plugin is supported by MMOProfiles.
</p>
