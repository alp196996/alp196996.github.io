# 👯‍♀️ Friends

<p>
  The friends system built into MMOCore is a simple yet effective system.
  When using the
  <code>
    /friends
  </code>
  command, it will pull up a GUI that shows all your current friends and
  options to add more.
</p>
<p>
  The stats shown when hovering over an online player include class, level,
  and online time. For an offline player, you will see how long they have
  been off!
</p>

<p>
  You can disable the friend module by simply removing the
  <code>
    /friends
  </code>
  command in the
  <code>
    commands.yml
  </code>
  config file
</p>
