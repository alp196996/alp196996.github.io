# 🦾 Modularity

<p>
  MMOCore is a modular plugin: it has specific modules you can choose to
  keep or to throw away. For instance, you can choose to use the default
  MMOCore party system, or choose another plugin - as long as it's supported
  by MMOCore - and the integration will be flawless. MMOCore provides the
  core features for your RPG server (classes, skills, leveling and professions)
  but it is definitely best used along with other RPG plugins like quest
  and guild solutions.
</p>
<p>
  Currently MMOCore has modules for quests, parties, guilds, skill creation.
  Modularity is great because you can still use the default MMOCore module
  if you want, while more advanced users can switch to their favourite and
  more advanced plugins, allowing for much more configurability
</p>
