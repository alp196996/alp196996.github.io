# 🔌 Other Plugins

<p>
  MMOCore only has MythicLib as hard dependency, but all of our soft dependencies
  are highly recommended for the RPG best experience.
</p>

## MMOItems

<p>
  This plugin is meant to go hand in hand with MMOCore. MMOItems provides
  item level &amp; class restrictions, Item Stats, Quest Objectives, Triggers,
  Drop Table Items, and tons of other stuff. MMOItems is the only plugin
  which has built-in support, and it can't be beat! You do not need to install
  any compatibility addon, all the following features are automatically setup
  when the plugins detect each other. More info on
  <a href="./mmoitems">
    this wiki page
  </a>
  .
</p>

## MythicMobs

<p>
  MythicMobs adds new quest objectives and allows you to create an infinite
  amount of skills for MMOCore. Moreover, MMOCore adds new drop table items
  to MythicMobs. More info on
  <a href="./mythic">
    this wiki page
  </a>
  .
</p>

## Fabled, ProSkillAPI, SkillAPI

<p>
  Since MythicLib 1.3 you can also register custom skills in MythicLib,
  MMOCore and MMOItems using the Fabled skill creation system. More info
  on
  <a href="./../../mythiclib/skills/custom/fabled">
    this wiki page
  </a>
  .
</p>

## Hologram Plugins

<p>
  We support a lot of hologram plugins to allow for special holograms/indicators
  built into the plugin, for instance when fishing or gaining main class/profession
  experience. More info on
  <a href="./../../mythiclib/features/compatibility#hologram-plugins">
    this wiki page
  </a>
  .
</p>

## WorldGuard, Residence

<p>
  WorldGuard and Residence are recommended in order to utilize flag checks
  and skills. WorldGuard can be used to restrict fishing drop tables and
  block regen/restrictions to certain regions. You can find the full WG flag
  list
  <a href="./../../mythiclib/features/compatibility#flag-plugins">
    here
  </a>
  .
</p>
