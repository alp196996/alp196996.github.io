# ⭐ Experience Boosters

<p>
  Admins can use a
  <a href="./../general/commands">
    command
  </a>
  to create EXP boosters that apply to all players on the server! Using
  the
  <code>
    /mmocore booster create
  </code>
  command, you can specify how much extra % EXP players will earn, and even
  <strong>
    which profession
  </strong>
  the booster applies for.
</p>

## Compatibility Options

<p>
  This type of EXP Boosters 100% EULA compliant. The best way to utilize
  these boosters is create items with this command bound to right clicks,
  in order to have consumables that apply EXP boosts for all players currently
  connected!
</p>
<p>
  The
  <code>
    /mmocore booster create
  </code>
  command has an optional
  <code>
    player
  </code>
  parameter that you can use to specify the booster creator. This way, other
  players can see who activated the EXP booster.
</p>
<p>
  When a booster is created, a message is broadcast in the entire server.
  It can be edited in the
  <code>
    messages.yml
  </code>
  config file.
</p>
