# 🔌 Installation Guide

<p>
  Many plugin issues are due to an invalid installations, so make sure you
  read the following tutorial to spare some time debugging!
</p>
<ol>
  <li>
    Make sure you install
    <a href="https://www.spigotmc.org/resources/mythiclib.90306/" target="_blank"
    rel="noreferrer">
      MythicLib
    </a>
    first
  </li>
  <li>
    Drag &amp; drop the downloaded MMOCore file in your plugin folder.
  </li>
  <li>
    Restart your server to generate default config files.
  </li>
  <li>
    <strong>
      UNINSTALL the MMOMana addon for MMOItems if you have it installed
    </strong>
    , it is not meant to be used alongside any plugin that already has mana
    (like MMOCore).
  </li>
  <li>
    MMOCore comes with default configs showcasing various compatibility features
    related to other plugins, including WorldGuard, Citizens, MythicMobs, MMOItems...
    If these plugins are not installed, you will get some startup warnings
    in your console, which is completely normal. Take your time to explore
    these features and install the required plugins if you want to use them.
  </li>
</ol>
