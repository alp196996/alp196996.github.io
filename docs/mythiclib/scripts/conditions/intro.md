# Introduction

<p>
  A script only executes its list of mechanics
  <strong>
    when all of its conditions are met
  </strong>
  . They are checked in-order, which means MythicLib will first evaluate
  the first condition, stop there if it's not met, then evaluate the second,
  stop if not met, etc... If all conditions are met, then mechanics are executed
  one after the other.
</p>
