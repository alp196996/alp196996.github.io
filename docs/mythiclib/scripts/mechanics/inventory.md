# 🎒 Inventory

## Close Inventory

<p>
  Use this mechanic to close the current player's inventory. This can be
  used inside any MMO plugin UI: MMOItems crafting stations, MMOProfiles
  profile selection, MMOCore skill tree UIs, etc...
</p>

## Go Gack

<p>
  Use this to have the player navigate to the last page visited. For instance,
  this can be used to go back from the
  <em>
    Class Confirmation UI
  </em>
  to the
  <em>
    Class Selection UI
  </em>
  in MMOCore for instance.
</p>
