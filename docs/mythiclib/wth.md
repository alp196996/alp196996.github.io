# 🤔 What is MythicLib?

<p>You might be wondering - <strong>I installed MMOItems or MMOCore, I already have enough plugins on my server, why do I need to install and worry about an additional plugin?</strong></p>

<p>MythicLib is a little more than just a library that contains shared code and utilities. It's a plugin that centralizes features to reduce config duplication and code redundancy.</p>
<p>It centralizes fundamental RPG features such as:</p>
<ul>
  <li>
    the MMO player stats system,
  </li>
  <li>
    the MMO damage and element system,
  </li>
  <li>
    damage and regen indicators,
  </li>
  <li>
    the MMO skill system and scripting engine,
  </li>
  <li>
    integrations with other plugins,
  </li>
  <li>
    many smaller-size shared config options,
  </li>
</ul>
